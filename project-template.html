Fleurish

UI/UX

Course

User Interface User Experience Design

Completed

Fall 2024

Instructor

Safa Niknami

Prompt: Use User Interface and User Experience design methods to create an app that addresses a real world problem in you experience in health, finance, education, or any other field.
Response: Fleurish is a game that helps users track how much water they drink, while encouraging consistent hydration. It’s perfect for anyone looking to improve their health or needing a reminder to drink more. Users can team up with friends and family to stay motivated as they grow their plants by meeting hydration goals.
Skills Learned: Formulating a plan, strategy and scope for creating an app and website. Creating user flow diagrams, design systems, site maps and wireframes. Creating interactive media compatible for all screen sizes. Manipulating form, color, type and imagery, hierarchy and grid systems to emphasize a desired user action.
